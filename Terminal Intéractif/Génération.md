# MSFVenom
## Payload
Une suite d'étapes séparés par un /
```<OS>/[architecture/]<bind|reverse><_|/><protocole>```
* OS
* Architecture
* Bind / reverse
*  Staged ou non ( / ou _ après bind ou reverse, dénotant une étape de moins )
* Protocole
